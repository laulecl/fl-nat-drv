Ce projet est un driver pour pouvoir utiliser le Native Instruments Maschine Mikro MK3 avec FL Studio en mode MIDI.

Il permet de contrôler :
- le rack de cannaux
- le mixer
- la playlist & les patterns
- les plugins (instruments & effets)
